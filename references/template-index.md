# Template Index

Keep one row per approved template in `templates/`. The goal is not to summarize every sentence; the goal is to make retrieval fast.

## How to Fill This Table

- `template_id`: a short stable ID you can cite in feedback
- `file_path`: the actual file under `templates/`
- `language`: Chinese, English, or mixed
- `use_case`: master's PS, PhD SOP, scholarship essay, exchange application, and so on
- `field`: the closest academic or professional field
- `profile_tags`: short tags for the applicant profile, separated by `;`
- `tone_tags`: short style tags, separated by `;`
- `best_for`: what this template is especially good at
- `notes`: anything worth remembering for retrieval
